## Tecnologías utilizadas

- Backend -> Firebase
- Frontend -> Angular, Ionic y capacitor
- Editor de codigo -> VSCode
- Documentación -> Compodoc
- Servicio de correo -> SMTP2GO
- Despliegue de la página web -> Netlify
- Diseño de las aplicaciones -> Figma
